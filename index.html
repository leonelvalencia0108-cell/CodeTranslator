1. Tecnología Propietaria (Code AI)

Todo el procesamiento de lenguaje natural, análisis de contexto y traducción se realiza a través de Code AI, el motor de inteligencia artificial neuronal desarrollado de forma nativa y exclusiva por CodeCompany. Al procesar tus solicitudes dentro de nuestra propia infraestructura, garantizamos la máxima privacidad e independencia tecnológica.

2. Recopilación y Uso de Datos

Los textos y comandos de voz que procesas a través de nuestra interfaz son enviados a la API de Code AI únicamente para su traducción en tiempo real. No dependemos de servicios externos, no almacenamos tus conversaciones en bases de datos, no vendemos información y no utilizamos tus datos para el entrenamiento de futuros modelos sin tu consentimiento explícito.

3. Procesamiento de Voz y Caché Local

La captura de voz está encriptada de extremo a extremo hacia nuestros servidores. Utilizamos el almacenamiento local de tu navegador (localStorage) estrictamente para guardar tus preferencias de interfaz (como el idioma seleccionado o el consentimiento de cookies) para que tu flujo de trabajo no se interrumpa.

1. Uso de Servicios y Límites

El servicio de CodeTranslator conecta directamente con los endpoints propietarios de Code AI. Su uso gratuito está sujeto a una política de uso justo (Fair Use). CodeCompany se reserva el derecho de restringir el acceso temporalmente en caso de detectar abuso, automatizaciones mediante bots no autorizados o ataques de sobrecarga a nuestra API nativa.

2. Naturaleza del Servicio y Precisión

La traducción es generada por modelos probabilísticos de lenguaje de Inteligencia Artificial. Aunque aplicamos instrucciones estrictas de razonamiento lingüístico para obtener resultados de calidad humana, el servicio se provee "tal cual". No ofrecemos garantías absolutas de exactitud, especialmente en contextos médicos o legales críticos.

3. Restricciones de Uso

Queda estrictamente prohibido usar la herramienta para evadir bloqueos de seguridad, traducir material ilegal, realizar ingeniería inversa a nuestros prompts de sistema, o integrarlo en aplicaciones comerciales de terceros sin un acuerdo de licencia empresarial con CodeCompany Services.
